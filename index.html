About Me

I'm a computer science major at Portland State University, as well as an active intern in the PSU/PDX Cooperative Education Program (PCEP).

My first rotation in PCEP is with Multnomah County, and I'm excited to be a part of their team of .NET developers creating software products for various county departments. I'm new to programming in C# and .NET but so far I'm enjoying the overall sense of integration that comes with working in a Microsoft environment.

As for this website, it's a quick little demo site I'm deploying for a web development class at PSU. If you recognize a lot of default Bootstrap styling, you have a sense of the time crunch I'm working under.
